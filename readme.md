## 설명
개인적인 목적으로 만든 Ubuntu bash shell용 설정 세트입니다.
이 세트에는 아래의 기능을 포함하고 있습니다.
* 예쁜 프롬프트
* git branch 이름 표시
* vim 설정
* 환영/안녕 메시지(figlet, toilet package 필요)
* man page를 예쁘게 꾸미기
등의 기능을 포함하고 있습니다.

## 설치 방법
```{r, engine='bash', count_lines}
$ git clone http://github.com/hulryung/shell.git
$ cd shell
$ sh install.sh
```
